# Installation

## Option A: Install from a built ZIP

1. Download the [latest Release](https://github.com/1jamesthompson1/sp-invoicer/releases/latest) ZIP file (`invoice-maker-vX.Y.Z.zip`).
2. Open Super Productivity and go to the plugin manager.
3. Choose **Install/Load Plugin from File**.
4. Select the downloaded ZIP and enable the plugin.

## Option B: Build it locally

1. Clone the repository:

```bash
git clone https://github.com/1jamesthompson1/sp-invoicer.git
cd sp-invoicer
```

2. Build the plugin:

```bash
npm run build
```

3. Use the generated ZIP file from the `dist/` directory and load it in Super Productivity via the plugin manager.
